# Software Development Factory
[![Status: Concept](https://img.shields.io/badge/Status-Beta-purple.svg)](https://github.com/rubipreethi/Software-Development-Factory)

A control-plane-driven, multi-agent AI system that translates Product Requirements Documents (PRDs) into deployed artifacts automatically.

## Quick Start
```bash
pip install -r requirements.txt
uvicorn src.api.main:app
```
**Access the dashboard at**: http://localhost:8000/

See `docs/PROJECT.md` for full architectural concepts, endpoints, schema references, and environment configuration instructions.
